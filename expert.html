Activity 1: Open and Close

In the open and close experiment using a combination of a servo motor and an ultrasonic sensor with an Arduino Uno, you create a dynamic system that responds to its surroundings.
The servo motor acts as the mechanism for opening and closing, while the ultrasonic sensor serves as the eyes, detecting the distance to an object. The Arduino Uno orchestrates the interaction by programming the servo motor to move based on the real-time feedback from the ultrasonic sensor.
For instance, you might design a setup where the servo motor opens a door or a gate when an object is detected within a certain range and closes it when the object moves away. This integration provides an interactive and automated solution, showcasing the versatility of combining different components in Arduino projects for practical applications.

Activity 2: Distance Measurement System with Audio and Visual Feedback

This project combines an Arduino Uno, an ultrasonic distance sensor (HC-SR04), a buzzer, and an LED to measure distances and provide feedback using both audio and visual indicators. The ultrasonic sensor calculates the distance to an object and triggers the LED and buzzer to signal the proximity of the object.

Author : Russel Electronics
Published Date : January 15, 2022

Code :
const int TRIG_PIN = 6;
const int ECHO_PIN = 7;
const int BUZZER_PIN = 3;
const int Led_Pin = 13;
const int DISTANCE_THRESHOLD = 50;

float duration_us, distance_cm;

void setup() {
Serial.begin (9600);
pinMode(TRIG_PIN, OUTPUT);
pinMode(ECHO_PIN, INPUT);
pinMode(BUZZER_PIN, OUTPUT);
}
void loop() {

digitalWrite(TRIG_PIN, HIGH);
delayMicroseconds(10);
digitalWrite(TRIG_PIN, LOW);

duration_us = pulseIn(ECHO_PIN, HIGH);
distance_cm = 0.017 * duration_us;

if(distance_cm < DISTANCE_THRESHOLD)
digitalWrite(BUZZER_PIN, HIGH);
else
digitalWrite(BUZZER_PIN, LOW);


Serial.print("distance: ");
Serial.print(distance_cm);
Serial.println(" cm");

delay(500);
}
